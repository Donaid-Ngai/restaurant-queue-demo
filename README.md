# Restaurant Queue Demo

A quick prototype for a restaurant waitlist flow built with Next.js and Chakra UI.

## What it includes

- Guest join form for QR-code visitors
- Restaurant dashboard for managing the queue
- Local browser storage for demo data persistence
- Resettable seeded sample data

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- This demo is intentionally client-side only.
- For a production version, the next step would be Supabase for persistence and auth, plus SMS or notification flows.
